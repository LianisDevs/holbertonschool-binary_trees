# BINARY TREES

* 0- creates a binary tree node

* 1- inserts node as the left child of another node

* 2- inserts node as the right child of another node

* 5- checks if given node is a root

* 7- goes through a binary tree using in-order traversal

* 8- goes through a binary tree using post-order traversal
